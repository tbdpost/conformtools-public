# conform.tools

Public binaries repository for conform.tools releases and auto-updates.

Download: [Latest Release](https://github.com/tbdpost/conformtools-public/releases)

Note: GitHub automatically exposes `Source code (zip)` and `Source code (tar.gz)` links for tags in public repositories. Those archives are generated from this binaries-only repository and do not contain the conform.tools application source code.

Use the attached release assets (`.dmg`, `.deb`, `.rpm`, `.AppImage`, `.msi`, `.exe`, `.flatpak`) as the official downloads.
